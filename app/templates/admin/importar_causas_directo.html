Importar Causas Directamente

Volver
Seleccionar archivo Excel
El archivo debe contener una columna llamada "Causa" o similar.
Instrucciones

Esta es una interfaz simplificada para importar causas de pérdida. Para usar la interfaz completa de importación, utilice el módulo de Datasets.

  1. Seleccione un archivo Excel que contenga las causas de pérdida
  2. El sistema intentará detectar automáticamente la columna con las causas
  3. Si se importan correctamente, será redirigido a la lista de causas

Formatos de Dataset
Formato para Variedades

El archivo debe contener las siguientes columnas:

  • FLOR: Nombre de la flor
  • COLOR: Color de la flor
  • VARIEDAD: Nombre de la variedad
FLOR COLOR VARIEDAD
NOVELTY YELLOW ZIPPO
NOVELTY GREEN BOTANI
DAISY RED VALENTINO
Formato para Bloques y Camas

El archivo debe contener las siguientes columnas:

  • BLOQUE: Identificador del bloque
  • CAMA: Número de cama
  • LADO: Lado de la cama (opcional)
BLOQUE CAMA LADO
1 1 A
1 1 B
1 2 A
Formato para Áreas

El archivo debe contener las siguientes columnas:

  • SIEMBRA: Identificador del tipo de siembra
  • AREA: Área en metros cuadrados
SIEMBRA AREA
MEDIA CAMA 12.5
CAMA COMPLETA 25.0
DOBLE CAMA 50.0
Formato para Densidades

El archivo debe contener la siguiente columna:

  • DENSIDAD: Descripción de la densidad de siembra
DENSIDAD
BAJA (1 planta/m²)
NORMAL (2 plantas/m²)
ALTA (4 plantas/m²)
